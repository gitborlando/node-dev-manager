# 02 Extreme Compact UI

## 目标

- 把主界面继续压缩到高密度、低层级布局。
- 让按钮同时承担状态显示与操作入口。
- 将日志区升级为更接近终端的展示。
- 默认窗口按最小可用尺寸启动。
- 清掉剩余 Tauri 壳层目录。

## 执行

1. 顶栏缩成单行，只保留抽屉开关、应用名和新建入口。
2. 移除独立状态胶囊、PID、更新时间、备注等次要状态区。
3. 标签页中的动作按钮直接显示当前状态语义：`启`、`停`、`忙`、`重`。
4. 日志头主按钮改成“状态 + 操作”一体，按运行态自动切换文案和配色。
5. 接入 `@xterm/xterm` 与 `@xterm/addon-fit`，替换原先纯文本日志列表。
6. 收紧抽屉、面板、输入框、按钮和全局留白，移除业务样式里的 `margin`。
7. Electron 启动尺寸改为 `820x560`，与最小尺寸一致。
8. 删除 `apps/desktop/src-tauri` 目录内残余文件。

## 校验

- `pnpm typecheck`
- `pnpm build`
