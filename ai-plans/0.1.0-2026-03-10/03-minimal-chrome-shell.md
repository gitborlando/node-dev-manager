# 03 Minimal Chrome Shell

## 目标

- 将主界面收敛为极简结构，顶部直接承载项目 tabs。
- 去掉最外层留白，避免窗口边缘出现多余抖动与卡片感。
- 将新建/编辑项目入口从侧边抽屉改为居中模态框。
- 关闭系统默认标题栏，改由应用自己托管最小化与关闭按钮。

## 执行

1. Electron 主窗口改为 `frame: false`，并通过 preload 暴露窗口最小化、关闭接口。
2. 顶部重构为 `品牌 + tabs + 新建 + 窗口控制` 的单行标题栏。
3. `ProjectTabs` 只保留项目标题，不再在 tab 内放运行和删除动作。
4. 移除主内容区最外层 `padding`，将日志面板直接铺满可用区域。
5. 项目表单由左侧抽屉切换为居中模态框，保留现有导入、保存、删除能力。
6. 全局高度切到 `100%/100vh` 并限制 `body` 滚动，降低内容略高于窗口时的抖动。

## 校验

- `pnpm typecheck`
- `pnpm build`
