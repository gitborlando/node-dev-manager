# 01 Shell Follow Up

## 目标

- 按最新 `plan.md` 收口自定义标题栏和 tab 条带的细节。
- 修复运行按钮 hover 状态跳变。
- 将项目模态框进一步压平，去掉 placeholder、边框和分割线。

## 执行

1. 标题栏最大化态补成三边安全内边距，减轻右上角窗口按钮被系统边缘遮挡的问题。
2. tab 条带切回拖拽区，tab 本身改成纯平样式，去掉圆角和边框并增加横向留白。
3. 标题栏品牌、tab 标题和日志区文字整体上调一档字号。
4. 运行按钮的状态色补专属 hover 规则，避免继承通用按钮 hover 后出现颜色跳变。
5. 模态框移除 placeholder、外边框、头尾分割线和表单描边，统一圆角到 `12px`。

## 校验

- `pnpm typecheck`
- `pnpm build`
